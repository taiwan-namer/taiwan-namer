# Spec: 更新網站頁尾版權文字

## Context & Goals (背景與目標)

目前網站頁尾 (Footer) 的版權年份為 2025。為保持資訊的時效性與專業性，並增加品牌識別，需要進行以下更新：
1.  將版權年份從 2025 更新為 2026。
2.  在版權文字後方加上品牌標語 " | Powered by AI Boss"，以強化品牌形象並增加一致性。
3.  此修改必須完全保持現有的視覺排版、樣式與響應式設計，不影響任何其他頁尾元素（如連結、社交圖示、Affiliate Disclosure 等）。

此修改符合專案憲法 **Core Principles I (User-First)** 與 **V (SEO & Shareability)**，透過維護專業外觀提升使用者信任與品牌識別。

## Proposed Changes (預計修改的檔案與邏輯)

此修改僅涉及單一檔案中的靜態文字內容。

**修改檔案：**
- `app/components/Footer.tsx`

**具體修改邏輯：**
1.  在 `Footer` 元件中，找到顯示版權年份與文字的段落 (`<p>` 標籤)。
2.  將文字內容從 `© 2025 台味命名大師` 更新為 `© 2026 台味命名大師 | Powered by AI Boss`。
3.  確保修改後的文字仍位於相同的 HTML 結構與 CSS 類名 (`text-zinc-300`) 之下，以維持現有的字體大小、顏色、間距與排版。

**範例程式碼區塊 (修改前後對比)：**
```tsx
// 修改前
<p className="text-zinc-300">
  © 2025 台味命名大師
</p>

// 修改後
<p className="text-zinc-300">
  © 2026 台味命名大師 | Powered by AI Boss
</p>
```

## Verification Plan (如何驗證)

1.  **視覺回歸測試：**
    - 在開發環境中執行 `npm run dev`，開啟網站並滾動至頁面底部。
    - 確認頁尾整體版面、間距、顏色與其他元素（如連結、Affiliate Disclosure 文字）與修改前完全一致，無任何視覺破壞。
    - 在不同螢幕寬度（桌面、平板、手機）下檢查頁尾的響應式排版是否正常。

2.  **內容正確性測試：**
    - 確認頁尾顯示的文字為 `© 2026 台味命名大師 | Powered by AI Boss`。
    - 確認年份正確為 `2026`。
    - 確認品牌標語 `Powered by AI Boss` 已正確顯示，且與前段文字以分隔符 `|` 連接。

3.  **建置檢查：**
    - 執行 `npm run build`，確認專案能成功建置，無任何 TypeScript 錯誤或警告。
    - 部署後，於正式環境再次執行上述視覺與內容檢查。
