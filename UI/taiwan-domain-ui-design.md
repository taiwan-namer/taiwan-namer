### 2.2 網域卡片設計細節

#### A. 卡片層級系統
```
🏆 最推薦 (推薦徽章)
├─ 視覺強化：金色邊框 + 陰影
├─ 優先顯示 .com
└─ CTA 按鈕使用對比色（橙色/綠色）

一般推薦
├─ 白色卡片 + 淺灰邊框
└─ CTA 按鈕使用次要色（藍色）
```

#### B. 卡片內容結構（由上而下）
1. **網域名稱** - 大字體，粗體
2. **英文拼音** - 小字體，灰色
3. **寓意說明** - 中等字體，帶 emoji 增加趣味
4. **價格標籤** - 醒目顯示，帶「起」字降低承諾
5. **主要 CTA** - 查價按鈕（1-2 個）
6. **次要操作** - 複製/收藏/分享（圖示按鈕）

#### C. CTA 按鈕優先級
```
主要按鈕（Primary）- 查價購買
├─ 大尺寸
├─ 高對比色（綠色/橙色）
└─ 明確動詞 + 利益點

次要按鈕（Secondary）- 複製/收藏
├─ 圖示 + 文字（或僅圖示）
├─ 灰色/淡色
└─ 不搶主 CTA 視覺
```

## 3. CTA 設計邏輯與文案

### 3.1 CTA 分類與優先級

#### Level 1 - 主轉換 CTA（最高優先級）
**目標：導向聯盟連結**

| 位置 | 文案 | 視覺 | 觸發事件 |
|------|------|------|----------|
| 網域卡片 | 🛒 GoDaddy 立即查價 | 橙色大按鈕 | outbound_click_godaddy |
| 網域卡片 | 💳 Namecheap 比價 | 綠色按鈕 | outbound_click_namecheap |
| 結果頁底部 | 🚀 推薦主機方案 - 首年 $2.95 | 全寬橫幅 | outbound_click_bluehost |

**文案原則：**
- 動詞開頭（立即、馬上、現在）
- 包含利益點（查價、比價、優惠）
- 降低風險（免費查詢、無需註冊）

#### Level 2 - 互動 CTA（培養參與）
**目標：增加停留時間與收藏率**

| 功能 | 文案 | 視覺 | 觸發事件 |
|------|------|------|----------|
| 複製網域 | 📋 複製 | 圖示按鈕 | domain_copied |
| 收藏網域 | ❤️ 收藏 | 圖示按鈕（可toggle） | domain_favorited |
| 分享結果 | 🔗 分享給朋友 | 圖示按鈕 | share_clicked |

#### Level 3 - 探索 CTA（引導深入）
**目標：提供更多價值，建立信任**

| 位置 | 文案 | 視覺 | 目的 |
|------|------|------|------|
| 結果頁中段 | 💡 查看網域挑選指南 | 文字連結 | 導向部落格 |
| 首頁底部 | 📚 閱讀架站教學 | 卡片連結 | 提供附加價值 |
| 導航列 | 關於我們 | 文字連結 | 建立信任 |

### 3.2 文案心理學應用

#### 損失厭惡（Loss Aversion）
```
❌ 避免：「購買網域」
✅ 使用：「別讓別人搶走這個好名字」
```

#### 社會認同（Social Proof）
```
❌ 避免：「立即購買」
✅ 使用：「1,200+ 人已購買類似網域」
```

#### 稀缺性（Scarcity）
```
❌ 避免：「查看價格」
✅ 使用：「限時優惠 - 首年 30% OFF」
```

#### 具體化價值（Concrete Value）
```
❌ 避免：「低價優惠」
✅ 使用：「首年只要 NT$ 300（原價 NT$ 450）」
```

### 3.3 CTA 測試變體（A/B Testing 建議）

#### 網域查價按鈕
```
變體 A：「🛒 GoDaddy 查價」
變體 B：「💰 查看 GoDaddy 優惠」
變體 C：「🔍 立即查詢可用性」
```

#### 主機推廣 CTA
```
變體 A：「🚀 查看推薦主機」
變體 B：「💻 網域買了，主機呢？」
變體 C：「⚡ 限時主機優惠 - 首年 $2.95」
```

### 4.4 資料結構定義
