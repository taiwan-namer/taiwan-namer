# 台味命名大師 (Taiwan Namer)

全台唯一！專懂台灣諧音梗、在地文化、算命筆畫的網域生成器。

## 技術棧

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Lucide React**

## 設計風格

- 極簡、深色模式 (Dark Mode)
- 漸層背景光暈 (Gradient Glows)
- 玻璃擬態 (Glassmorphism) 卡片
- 清晰易讀字體 (Noto Sans TC)

## 執行方式

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 在瀏覽器開啟 [http://localhost:3000](http://localhost:3000)

## 建置

```bash
npm run build
npm start
```
